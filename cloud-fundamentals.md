# Cloud Computing Basics

These notes summarize the cloud fundamentals I studied while learning AWS and Linode.

## What is Cloud Computing?

Cloud computing means using servers, storage, and networking over the internet instead of managing physical hardware locally.

**Advantages**
- Pay only for what you use
- Easy to scale resources
- No need to manage physical servers

## Cloud Service Models

**IaaS (Infrastructure as a Service)**  
Provides virtual machines and infrastructure.  
Example: AWS EC2, Linode.

**PaaS (Platform as a Service)**  
Platform for developers to deploy applications without managing servers.

**SaaS (Software as a Service)**  
Software accessed through the internet.  
Example: Gmail, Google Docs.

## Cloud Server Deployment

I learned how to deploy a virtual server using Linode.

Basic steps:
1. Create a cloud server
2. Select region and operating system
3. Configure SSH access
4. Connect to the server remotely
