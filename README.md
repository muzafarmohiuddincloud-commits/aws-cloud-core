# Day 1 – AWS Basics & Account Setup

## What I learned
- What is cloud computing
- Overview of Amazon Web Services (AWS)
- Difference between Root user and IAM user
- Importance of security in AWS accounts

## AWS Account Setup
- Created an AWS free tier account
- Completed email and phone verification
- Understood free tier limits

## IAM (Identity and Access Management)
- Learned why root user should not be used for daily work
- Created an IAM admin user
- Assigned AdministratorAccess policy
- Enabled Multi-Factor Authentication (MFA) using Google Authenticator
- Successfully logged in using IAM admin user

## Security Best Practices
- Root user reserved only for billing and emergency access
- IAM users for daily operations
- Enabled MFA for better account protection

- GitHub
